# chicago_crime_analysis

SQL-based data analysis project using Chicago public datasets.

## Overview
This project uses Python, SQLite, and SQL in Jupyter Notebook to analyze:
- Chicago Crime Data
- Chicago Census Data
- Chicago Public Schools Data

## Database
SQLite database: `FinalDB.db`

Tables:
- census_data
- chicago_public_schools
- chicago_crime_data

## Key Analysis
- Total number of crimes
- Most crime-prone community area
- Average safety score by school type
- Community with highest hardship index

## Tech Stack
Python | Pandas | SQLite | SQL | Jupyter Notebook
